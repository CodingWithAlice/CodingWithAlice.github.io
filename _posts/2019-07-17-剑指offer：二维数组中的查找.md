# 二维数组中的查找（javascript）



### 方法一：暴力遍历

```javascript
function Find(target, array)
{
    for(let i = 0;i<array.length;i++){
        for(let j=0;j<array[0].length;j++){
            if(array[i][j] == target){
                return true
            }
        }
    }
}
```
