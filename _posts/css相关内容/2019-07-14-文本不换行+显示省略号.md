# 显示省略号

```css
/* 一行文本省略 */
white-space: nowrap;
text-overflow: ellipsis;
overflow: hidden;
```

```css
/* 多行文本省略(ps：此处是两行) */
text-overflow: -o-ellipsis-lastline;
overflow: hidden;
text-overflow: ellipsis;
/* display: -webkit-box 必须结合的属性 ，将对象作为弹性伸缩盒子模型显示 */
display: -webkit-box;
/* line-clamp 用来限制在一个块元素显示的文本的行数 */
-webkit-line-clamp: 2;
line-clamp: 2;
/* box-orient 必须结合的属性 ，设置或检索伸缩盒对象的子元素的排列方式 */
-webkit-box-orient: vertical;
```
